# EZ360PM Brand Guide

This application uses the supplied EZ360PM artwork and treats
`static/css/do not use.css` as a visual reference only. The legacy stylesheet is
not loaded by the application.

## Core palette

| Token | Value | Use |
| --- | --- | --- |
| Brand blue | `#1f5fbf` | primary actions, links, and focus states |
| Dark blue | `#184c9b` | link and blue-state contrast |
| Brand green | `#38b26f` | primary-action hover and positive accents |
| Dark green | `#2a8a55` | active-action contrast |
| Navy | `#111c32` | headings and high-emphasis text |

The canonical brand direction is blue to green. Primary buttons use an animated
blue-to-green sweep on hover and keyboard focus. Users who request reduced motion
get the same state change without animation.

## Asset use

| Context | Asset |
| --- | --- |
| Application header and sign-in | `images/ez360pm_logo.svg` |
| Compact mobile header | `images/EZ360PM_icon_transparent_128.png` |
| Browser favicon | `images/favicon.ico` |
| Apple touch icon | `images/EZ360PM_icon_white_256.png` |
| Installed web-app icons | white-background 256 px and 512 px icons |

Company-uploaded logos remain company identity and appear separately from the
EZ360PM product mark.
